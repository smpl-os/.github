## What is smplOS?

smplOS is a minimal Arch Linux distro built around one idea: **simplicity**.

It started as an attempt to build a lighter version of Omarchy — same keybindings, same themes, but without the bloat. That contribution was rejected, so we forked and built our own thing. What came out the other side isn't Omarchy lite — it's a different OS.

- 🪶 **< 800 MB RAM** at idle with the full desktop running
- ⚡ **Offline install in under 2 minutes** — no internet required
- 🎨 **14 built-in themes** — one command recolors every app on the desktop
- 🧩 **Modular editions** — Productivity, Creators, Communication, Development, AI
- 🌐 **Sandboxed web apps** — any site becomes an isolated native-feeling app
- 🔄 **Cross-compositor** — same configs, themes, and keybindings on Hyprland and DWM
- 🔧 **Migration system** — safe, versioned upgrades that never break configs

## Repositories

| Repo | Description | Language |
|------|-------------|----------|
| [smplos](https://github.com/smpl-os/smplos) | The OS itself — scripts, configs, themes, migrations, ISO builder | Shell |
| [smpl-apps](https://github.com/smpl-os/smpl-apps) | Native GUI apps: start menu, notifications, settings, app center, calendar | Rust |
| [nemo-smpl](https://github.com/smpl-os/nemo-smpl) | Nemo fork with live CSS theming, sidebar shortcuts, keybinding tooltips | C |
| [st-smpl](https://github.com/smpl-os/st-smpl) | st terminal patched for Wayland with SIXEL inline images | C |
| [eww](https://github.com/smpl-os/eww) | EWW fork powering the bar, quick-settings panel, and system tray | Rust |
